These are some scipts I use to connect to walle.

1. First, run `conn_walle.sh` on the laptop
2. Then SSH into Walle (192.168.50.1)
3. Run `conn_parrent.sh` on the pi from the ssh terminal

These steps will allow ssh from parent to host, as well as allow the pi to forward internet traffic through the parrent to the internet. Yay, git now works.
